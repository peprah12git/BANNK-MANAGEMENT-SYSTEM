Bank Management System
Description

A simple application that manages basic banking operations such as account creation, deposits, withdrawals, and balance checks. It helps automate customer data handling and ensures accurate banking transactions.

Features

Create accounts - Register new bank accounts of customers

View Accounts - Display all account within the details

Process transactions - Deposit and withdraw money

View transactions - Display Transaction history of an account

simpple Menu- Navigate through options

Technologies

Language: Java

Tools:IntelliJ 

Git & GitHub

How to Run

Clone the repository

Open in your IDE

Run the main file (main.py / Main.java, etc.)
